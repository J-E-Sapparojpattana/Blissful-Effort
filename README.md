# Blissful-Effort

# 🏛️ The Code of Silence
**"Success is the fruit of intentional action, not the gift of fate."**

The Code of Silence คือโปรเจกต์ Web-based Interactive ที่รวบรวม **"อุดมมงคล ๓๘ ประการ"** และหลัก **"วิริยะวาที/กรรมวาที"** (The Path of Diligence) เพื่อเป็นเข็มทิศในการดำเนินชีวิตที่ประณีต เรียบง่าย และทรงพลังในยุคดิจิทัล

## 🕯️ Philosophy (ปรัชญาของโครงการ)
ในโลกที่วุ่นวาย เราเชื่อใน **"ความสงบที่เป็นพลัง"** และ **"การลงมือทำที่เปลี่ยนชีวิต"** โปรเจกต์นี้ถูกออกแบบด้วยแนวคิด **Artisan Coding** ที่เน้นความประณีต (Craftsmanship) และความสงบ (Calm Technology)
* **Viriya-Vadi:** เน้นย้ำพลังแห่งความเพียรพยายาม
* **Kamma-Vadi:** เน้นย้ำว่าการกระทำคือเหตุปัจจัยของความสำเร็จ
* **Minimalism:** ตัดทอนสิ่งที่ไม่จำเป็น เพื่อให้เข้าถึงแก่นธรรมได้ชัดเจนที่สุด

## ✨ Features
- **38 Great Blessings:** เนื้อหาอุดมมงคลครบถ้วนทั้งภาษาไทยและอังกฤษ
- **Diligence First:** ส่วนการเรียนรู้เรื่องความเพียรที่เป็นจุดเริ่มต้นของทุกความสำเร็จ
- **Artisan Dark Mode:** ดีไซน์ที่สบายตา เหมาะสำหรับการอ่านและพิจารณาธรรมยามค่ำคืน
- **Clean Architecture:** เขียนด้วย Vanilla HTML/CSS/JS ไร้ Library ภายนอก เพื่อความเบาและยั่งยืน

## 🛠️ Usage
1. Clone repository นี้
2. เปิดไฟล์ `index.html` ผ่าน Browser ใดก็ได้
3. พิจารณาเนื้อหาทีละ Step เพื่อการฝึกฝนตนเองอย่างเป็นลำดับขั้น
